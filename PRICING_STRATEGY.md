# BG Remover 定价策略 v3.0（优化版）

## 1. 成本分析

### Remove.bg API 成本
- 官方 Free：50/月（免费）
- 官方 Starter：$9.99/月 = ¥70/月（100 次）= ¥0.70/次
- 官方 Pro：$39/月 = ¥273/月（500 次）= ¥0.546/次
- 官方 Enterprise：按量计费，约 ¥0.40-0.50/次

**我们的成本：** 按 remove.bg Starter 计算，约 ¥0.70/次

### 定价逻辑
```
API 成本：¥0.70/次
目标利润率：100-150%（即用户支付 ¥1.40-1.75/次）

积分包定价：¥1.50/次（利润 114%）
Pro 订阅：¥0.29/次（利润 -59%，但通过量大降低成本）
```

**关键：** Pro 订阅看似亏本，但实际上：
1. 用户不会每月都用满 100 次
2. 平均使用率约 60-70%
3. 实际成本 ¥0.42-0.49/次
4. 实际利润 40-50%

## 4. 成本对比分析

### 用户成本对比

#### 场景 1：轻度用户（月均 10 次）
```
积分包：10 × ¥1.50 = ¥15
Pro：¥29（浪费 90 次）
推荐：积分包
```

#### 场景 2：中度用户（月均 50 次）
```
积分包：50 × ¥1.40 = ¥70
Pro：¥29（节省 ¥41）
推荐：Pro
```

#### 场景 3：重度用户（月均 150 次）
```
积分包：150 × ¥1.20 = ¥180
Pro：¥29（节省 ¥151）
Pro Plus：¥79（节省 ¥101）
推荐：Pro Plus
```

#### 场景 4：超重度用户（月均 400 次）
```
积分包：400 × ¥1.20 = ¥480
Pro Plus：¥79（节省 ¥401）
推荐：Pro Plus
```

### 公司成本对比

#### 积分包用户（平均 50 次/月）
```
用户支付：50 × ¥1.40 = ¥70
API 成本：50 × ¥0.70 = ¥35
利润：¥35（50%）
```

#### Pro 用户（平均使用率 60%）
```
用户支付：¥29
API 成本：100 × 60% × ¥0.70 = ¥42
亏损：¥13（-45%）
但考虑：
- 用户粘性高
- 续费率高
- 可交叉销售
- 年付用户利润为正
```

#### Pro Plus 用户（平均使用率 70%）
```
用户支付：¥79
API 成本：300 × 70% × ¥0.70 = ¥147
亏损：¥68（-86%）
但考虑：
- 高价值用户
- 续费率最高
- API 访问产生额外收入
- 年付用户利润为正
```

**优化策略：**
1. 鼓励年付（年付用户利润为正）
2. 积分包用户是主要利润来源
3. 订阅用户是长期价值来源
4. 通过 API 访问产生额外收入

## 10. FAQ（常见问题）

### 积分包相关
**Q: 积分包有效期是多久？**
A: 365 天。购买后 365 天内可以使用，过期自动清零。

**Q: 积分包可以转账给其他用户吗？**
A: 不可以，积分包仅限购买者本人使用。

**Q: 如果我购买了积分包，后来又升级到 Pro，积分会怎样？**
A: 积分不会清零，会继续保留。升级到 Pro 后，优先使用订阅次数，积分作为备用。

**Q: 积分包支持退款吗？**
A: 支持 7 天无理由退款。

### 订阅相关
**Q: Pro 和 Pro Plus 的区别是什么？**
A: Pro 适合中度用户（100 次/月），Pro Plus 适合重度用户（300 次/月）。Pro Plus 还支持更高分辨率和更多批量处理。

**Q: 我可以先试用 Pro 版吗？**
A: 可以！新用户升级 Pro 后享受 7 天免费试用，无需绑定支付方式。

**Q: 如果我没用完本月的次数，下个月会清零吗？**
A: 是的，每个月 1 号重置。但 Pro 用户可以在个人中心查看使用统计。

**Q: 支持哪些支付方式？**
A: 目前支持支付宝、微信支付。后期计划支持 PayPal。

**Q: 如何取消订阅？**
A: 在个人中心随时可以取消，不会收取后续费用。

**Q: 可以退款吗？**
A: 支持 7 天无理由退款。

### 功能相关
**Q: 免费版和 Pro 版的区别是什么？**
A: 免费版注册赠送 3 次，2K 分辨率；Pro 版每月 100 次，4K 分辨率，还支持自定义背景色和批量处理。

**Q: 支持哪些图片格式？**
A: 支持 JPG、PNG、WebP，最大 10MB。

**Q: 处理速度有多快？**
A: 通常 1-3 秒，Pro 用户享受优先队列，速度更快。

**Q: 可以用于商业用途吗？**
A: 可以，所有版本都支持商业使用。

### API 相关
**Q: API 成本是多少？**
A: Pro 版及以上支持 API 访问，按实际调用次数计费，成本约 ¥1.50/次（与积分包相同）。

**Q: 如何获取 API 密钥？**
A: 升级到 Pro 后，在个人中心 → API 管理 → 生成密钥。

**Q: API 有速率限制吗？**
A: Pro 版 100 次/月，Pro Plus 版 300 次/月。
