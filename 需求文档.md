## 1. 产品概述

### 1.1 产品名称
Image Background Remover（暂定，待定品牌名）

### 1.2 产品定位
一款轻量级在线图片背景去除工具，用户上传图片后一键获得透明背景的 PNG 文件，无需注册，即用即走。

### 1.3 目标用户
- 电商卖家（商品主图抠图）
- 设计师（快速处理素材）
- 普通用户（证件照、头像处理）
- 海外用户为主（英文界面）

### 1.4 核心价值
- **快**：上传即处理，秒出结果
- **准**：Remove.bg API 保证抠图质量
- **轻**：无需注册，无广告干扰，纯净体验

## 2. 技术架构

### 2.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Next.js + TailwindCSS | 静态导出，部署至 Cloudflare Pages |
| 后端 | Cloudflare Workers | API 代理，转发请求至 Remove.bg |
| AI引擎 | Remove.bg API | 背景去除核心能力 |
| 存储 | 纯内存（不持久化） | 图片处理完即丢弃，不落库 |
| 部署 | Cloudflare（全站） | Pages + Workers，零运维 |

### 2.2 请求流程
```
用户上传图片
  → 前端（Cloudflare Pages）
  → POST /api/remove-bg
  → Cloudflare Worker（代理层）
  → 转发 multipart/form-data
  → Remove.bg API
  → 返回透明 PNG（二进制）
  → Worker 直接返回给前端
  → 前端展示结果 + 提供下载
```

### 2.3 隐私设计
- 图片不上传到任何自有服务器
- Worker 仅做透传，不记录、不存储图片内容
- Remove.bg 的数据处理遵循其隐私政策

## 3. 功能需求

### 3.1 核心功能（MVP 必做）

**F1 — 图片上传**
- 支持拖拽上传
- 支持点击选择文件
- 支持格式：JPG、PNG、WebP
- 文件大小限制：≤ 10MB（Remove.bg 免费版限制）
- 上传后即时预览原图

**F2 — 背景去除**
- 点击"Remove Background"按钮触发处理
- 显示 Loading 状态（进度条或 Spinner）
- 调用 Cloudflare Worker → Remove.bg API
- 错误处理：API 失败/超时/额度耗尽时显示友好提示

**F3 — 结果展示**
- 左右对比展示：原图 vs 去背结果
- 去背结果使用棋盘格背景表示透明区域
- 支持缩放预览（可选）

**F4 — 下载**
- 一键下载透明背景 PNG
- 文件名：`removed-bg-{原文件名}.png`

### 3.2 辅助功能（MVP 可选）

**F5 — 背景替换（可选）**
- 提供几种预设纯色背景（白、黑、蓝、红）
- 用户可自选颜色
- 下载时合成背景色输出

**F6 — 使用次数提示（可选）**
- 页面展示"今日剩余免费次数"（前端本地 localStorage 计数）
- 超出后引导用户刷新或明日再来

## 6. API 设计

### 6.1 Worker 接口

**POST /api/remove-bg**

- 请求：`Content-Type: multipart/form-data`，Body: `image_file (binary)`
- 响应（成功）：`Content-Type: image/png`，Body: 透明背景 PNG 二进制数据
- 响应（失败）：
```json
{
  "error": "错误信息",
  "code": "RATE_LIMIT | INVALID_FILE | API_ERROR"
}
```
